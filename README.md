YoutubeDl

Uses python 3
Tool that has for purpose to download a playlist from youtube 
Uses the library pafy : https://pythonhosted.org/Pafy/

To install it :
Clone it
Install python 3 
pip3 install pafy 
pip3 install youtube-dl
pip3 install pyqt5

run youtubedl.py

mainwindow.ui is the graphical interface open it using QTCreator
